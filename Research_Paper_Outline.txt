2. ABSTRACT
--------------------------------------------------------
In an era of hyper-information, decision-makers are often overwhelmed by fragmented data from news outlets, financial markets, and specialized sensors. This paper presents Global News AI, an interactive "World Monitor" dashboard designed to provide high-density situational awareness through a military-grade Spec Ops HUD (Heads-Up Display). By synthesizing real-time data from web scraping, cybersecurity feeds, aviation APIs, and climate sensors, the system creates a unified tactical view. A core novelty of this work is the SENTINEL-7 intelligence layer, which utilizes Large Language Models (LLMs) via the Groq Cloud API to generate contextual narratives and risk assessments from raw data. We discuss the system's decoupled architecture, the use of glassmorphism for cognitive load reduction, and the performance benefits of a custom intelligence caching layer.

3. INTRODUCTION
--------------------------------------------------------

* Problem: Existing news aggregators are too general, while professional OSINT tools are too complex for rapid situational awareness.
* Objective: To bridge the gap by creating an immersive, AI-augmented "Command Center" that visualizes global threats in real-time.
* Contributions: 
    1. A unified multi-domain intelligence pipeline (Finance, Tech, Climate, Conflict).
    2. An immersive 3D/2D geospatial visualization system.
    3. AI-driven automated report generation for abstract data points.

4. SYSTEM ARCHITECTURE
--------------------------------------------------------
The system employs a Decoupled Client-Server Architecture to ensure scalability and real-time responsiveness.

A. Data Ingestion Layer (The Sensors)
* Web Scrapers: Custom fault-tolerant scrapers (Cheerio) monitor financial indices (NIFTY 50) and maritime logistics.
* Public APIs: Integration with OpenSky (Aviation), Open-Meteo (Climate), and GNews (Media).
* Security Feeds: Real-time malware and IP threat intelligence from MalwareBazaar and Abuse.ch.

B. Intelligence Processing (The Brain)
* SENTINEL-7 Engine: Powered by Llama 3.3 (70B) via Groq Cloud API. It transforms raw numbers (e.g., a 2% drop in Suez Canal traffic) into actionable intelligence reports.
* Intel Cache Layer: An in-memory caching system with a 3-minute Time-To-Live (TTL) ensures the dashboard remains performance-responsive despite high-frequency data updates.

C. Visualization Layer (The HUD)
* Immersive Mapping: Built using react-globe.gl (Three.js) for 3D global context and Leaflet for 2D tactical precision.
* HUD Design: Utilizes Vanilla CSS with glassmorphism to create a non-intrusive, "heads-up" experience that maintains focus on the data.

5. KEY TECHNICAL INNOVATIONS
--------------------------------------------------------
I. Hybrid Data Synthesis
Unlike static dashboards, Global News AI triggers logic-based alerts. For example, the system cross-references real-time temperature data with geopolitical hotspots to flag potential "climate-driven instability."

II. Designing for High Density
The "Satellite Downlink" feature integrates 6+ concurrent live video streams (YouTube IFrame API). This requires careful DOM management to prevent browser memory leaks while ensuring zero-latency "live" surveillance.
